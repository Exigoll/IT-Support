Задача № 6. Вопрос об if.

Какие из перечисленных ниже alert( ) выполнятся? Какие конкретно значения будут результатами выражений в условиях if ( )?

						
// Вариант № 1
if (-1 || 0) {
  alert('first')
}
	
// Вариант № 2
if (-1 && 0) {
  alert('second')
}
	
// Вариант № 3
if (null || -1 && 1) {
  alert('third')
}

№ 1. true -> выполнится, т.к. оператор || возвращает значение true, если один из аргументов true;

№ 2. false -> не выполнится, т.к. оператор && возвращает значение true, в случае если оба аргумента true. В данном случае возвращается последнее значение, то есть 0, а 0 -> false;

№ 3. true -> выполнится, т.к.: сперва произойдет вычисление выражения -1 && 1 -> true; затем null || 1 -> true, т.к. оператор || возвращает значение true, если один из операндов true;

Задача № 27. Switch ( ) case.

Переменная num может принимать значения 1, 2, 3 или 4. Если она имеет значение 1, то выведите в консоль - 'зима', если 2, то - 'лето' и т.д. Если в переменной будет какое-то другое значение - выведите в консоль сообщение об ошибке. Решите задачу через switch ( ) case.

						
let num = 1
switch (num) {
  case 1 :
    console.log('зима');
  break;
  case 2 :
    console.log('весна');
  break;
  case 3 :
    console.log('лето');
  break;
  case 4 :
    console.log('осень');
  break;
  default :
    console.log('Ошибка')
  break
}
