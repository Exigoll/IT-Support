Task № 1. Шаблонные строки.

Что выведет этот скрипт?

						
let name = 'Andrew' 
console.log(`hello ${1}`)
console.log(`hello ${'name'}`)
console.log(`hello ${name}`)
						
					
							
let name = 'Andrew'
console.log(`hello ${1}`) // 'hello 1'
console.log(`hello ${'name'}`) // 'hello name'
console.log(`hello ${name}`) // 'hello Andrew'

Task № 2.

Даны переменные со значениями '5px' и '6px'. Найдите сумму пикселей из значений этих переменных и выведите ее на экран.

							
let one = '5px'
let two = '6px'
let result = parseFloat(one) + parseFloat(two)
console.log(result)

Task № 3.

Даны переменные со значениями '5.5px' и '6.25px'. Найдите сумму пикселей из значений этих переменных и выведите на экран.

							
let one = '5.5px'
let two = '6.25px'
let result = parseFloat(one) + parseFloat(two)
console.log(result)

Task № 4.

Модифицируйте предыдущую задачу так, чтобы к выводящемуся результату добавлялись буквы 'px'. То есть, если сумма равна 11.75, то на экран пусть выведется '11.75px'.

							
let one = '5.5px'
let two = '6.25px'
let result = parseFloat(one) + parseFloat(two)
console.log(result + 'px')
