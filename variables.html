Task № 1. Количество секунд.

Выведите в консоль следующие значения:

1. Количество секунд в сутках

2. Количество секунд в 30 сутках

3. Количество секунд в году

4. Количество минут в сутках

5. Количество минут в году

							
const minute = 60
const hour = minute * 60
const day = hour * 24
const week = day * 7
const mounth = day * 30
const year = mounth * 12

// № 1
console.log(day + ' seconds')  // result 86400 seconds

// № 2
console.log(day * 30 + ' seconds in 30 days')  // result 2592000 seconds in 30 days

// № 3
console.log(year + ' second in year')  // result 31104000 second in year

// № 4
console.log(day / 60 + ' minutes in day')  // result 1440 minutes in day

// № 5
console.log(day * 365 / 60 + ' minutes in year')  // result 525600  minutes in year

Task № 3. Площадь круга.

Переменная radius содержит значение с радиусом круга равно 10. По соответствующей формуле найдите площадь круга и запишите ее в переменную square. Выведите содержимое этой переменной на экран.

							
let radius = 10 // Радиус круга 
const π = 3.14 // константа, которая выражает отношение длины окружности к диаметру, всегда равна 3.14

let square = π * radius ** 2 // формула расчете s = π * r в квадрате
console.log(square) // 314

Task № 4. Площадь прямоугольника.

Переменные width и height содержат значения равные сторонам прямоугольника. Найдите площадь прямоугольника и запишите ее в переменную square. Выведите содержимое этой переменной в консоль.

							
let width = 10
let height = 15
let square = width * height
console.log(square) // 150

Task № 5. Площадь квадрата.

Переменная width содержит значение равное стороне квадрата. Найдите площадь квадрата и запишите ее в переменную square. Выведите содержимое этой переменной в консоль.

							
let width = 10
let square = width * width
console.log(square) // 100

Task № 6. Периметр прямоугольника.

Переменные width и height содержат значения равные сторонам прямоугольника. Найдите периметр прямоугольника и запишите его в переменную perimeter. Выведите содержимое этой переменной в консоль.

							
let width = 10
let height = 4
let perimeter = (width + height) * 2  // сумма длины и ширины умноженая на 2
console.log(perimeter) // 28

Task № 7. Из Цельсия в Фаренгейты.

Переменная tc содержит в себе значение с температурой, в градусах Цельсия. По соответствующей формуле выполните перевод этой температуры в градусы Фаренгейта.
